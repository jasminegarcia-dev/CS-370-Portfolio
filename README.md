CS 370 – Pirate Intelligent Agent Project

This repository contains my Deep Q-Learning implementation for the Treasure Maze project. I was given the maze environment, the TreasureMaze and GameExperience classes, and the neural network template. My main responsibility was implementing the reinforcement learning logic inside the qtrain function. I built the training loop using epsilon-greedy exploration, experience replay, and a target network. After adjusting hyperparameters like the number of epochs and memory size, I was able to reach a 100% win rate and pass the completion check.

This project connects directly to the broader field of computer science because it demonstrates how algorithms and AI systems are used to solve decision-making problems. Computer scientists design, test, and refine systems that can learn from data and improve over time. Reinforcement learning is especially relevant in areas like robotics, automation, and game AI.

When approaching a problem as a computer scientist, I break it down into smaller components, test incrementally, and use performance metrics to guide improvements. In this project, observing win rate trends and adjusting training parameters were essential to success.
